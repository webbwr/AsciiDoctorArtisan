# Ideas Folder

## What This Is

We plan new things here.

First we write ideas. Then we build them.

## What's Inside

```
openspec/
├── changes/      (New ideas)
├── archive/      (Done ideas)
└── README.md     (This file)
```

See `SPECIFICATIONS_HU.md` for all the rules.

## Add an Idea

### Step 1: Copy

Copy the template folder to your idea name.

### Step 2: Fill Files

- `proposal.md` - Why
- `tasks.md` - Work
- `design.md` - How
- `specs/` - Rules

### Step 3: Get OK

- Show it
- Talk
- Fix
- Get OK

### Step 4: Build

- Do work
- Check off
- Test

### Step 5: Save

- Move to `archive/`
- Update rules
- Close notes
