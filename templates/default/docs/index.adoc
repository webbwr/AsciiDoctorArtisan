== Introduction

This is the main documentation page for your AsciiDoc Artisan project. AsciiDoc is a lightweight markup language for authoring notes, articles, documentation, books, and more.

== Why AsciiDoc?

AsciiDoc offers several advantages:

* *Readability*: Plain text format that's easy to read and write
* *Flexibility*: Convert to multiple formats (HTML, PDF, EPUB, DocBook)
* *Power*: Rich feature set for technical documentation
* *Version Control*: Works seamlessly with Git

== Basic Syntax

=== Text Formatting

You can use *bold*, _italic_, `monospace`, and ^superscript^ or ~subscript~ text.

=== Lists

Unordered list:

* First item
* Second item
** Nested item
** Another nested item
* Third item

Ordered list:

. First step
. Second step
. Third step

=== Code Blocks

AsciiDoc supports syntax-highlighted code blocks:

[source,python]
----
def hello_world():
    """A simple Python function."""
    print("Hello, World!")
    return True

if __name__ == "__main__":
    hello_world()
----

[source,javascript]
----
function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet("AsciiDoc");
----

== Advanced Features

=== Include Files

You can break your documentation into multiple files and include them:

[source,asciidoc]
----
include::chapter1.adoc[]
include::chapter2.adoc[]
----

=== Custom Attributes

Define custom attributes for reusable content:

:product-name: AsciiDoc Artisan
:version: 1.0

This is {product-name} version {version}.

=== Conditional Content

You can show/hide content based on attributes:

[source,asciidoc]
----
ifdef::production[]
Production-only content
endif::[]

ifndef::draft[]
Non-draft content
endif::[]
----

== Building Your Documentation

=== HTML Output

To generate HTML output:

[source,bash]
----
asciidoctor docs/index.adoc -o output/index.html
----

=== PDF Output

To generate PDF output (requires asciidoctor-pdf):

[source,bash]
----
asciidoctor-pdf docs/index.adoc -o output/index.pdf
----

=== Watch Mode

For continuous building during development:

[source,bash]
----
# Using a file watcher
while inotifywait -e modify docs/*.adoc; do
    asciidoctor docs/index.adoc -o output/index.html
done
----
