== Endpoints

=== Users

==== List Users

`GET /users`

Returns a list of all users.

.Request
[source,http]
----
GET /users HTTP/1.1
Host: api.example.com
Authorization: Bearer <token>
----

.Response
[source,json]
----
{
  "data": [
    {
      "id": "1",
      "name": "John Doe",
      "email": "john@example.com"
    }
  ],
  "total": 1
}
----

==== Get User

`GET /users/{id}`

Returns a single user by ID.

.Parameters
[cols="1,1,3"]
|===
|Parameter |Type |Description

|id
|string
|User ID (required)
|===

==== Create User

`POST /users`

Creates a new user.

.Request Body
[source,json]
----
{
  "name": "Jane Doe",
  "email": "jane@example.com"
}
----

.Response (201 Created)
[source,json]
----
{
  "id": "2",
  "name": "Jane Doe",
  "email": "jane@example.com",
  "created_at": "2025-01-01T00:00:00Z"
}
----
