## Quick Start

New to AsciiDoc Artisan? Read these in order:

1. **how-to-install.md** - Get it on your computer
2. **how-to-use.md** - Learn how to use it
3. Start writing!

Want to help make it better?
4. **how-to-contribute.md** - Learn how to make changes
