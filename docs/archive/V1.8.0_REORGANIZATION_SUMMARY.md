## Rationale

### Why Reprioritize?

**User Value First:**
- Find/Replace is the most requested feature
- Spell checking is expected in every document editor
- These features provide immediate value to all users

**Advanced Features Second:**
- Auto-complete is powerful but not essential
- Syntax checking helps but spell checking is more important
- These features primarily benefit power users

**Quality Over Speed:**
- Smaller v1.8.0 scope = better quality (6-8 weeks vs 4-6 months)
- 100% test coverage maintained (90+ tests)
- Focused releases are easier to ship and support
